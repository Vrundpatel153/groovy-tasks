# Groovy Tasks

A root collection of daily coding tasks and projects.

## Structure

Projects are grouped by day:

- `Day-1/`
- `Day-2/`
- `Day-3/`
- `Day-4/`
- `Day-5/`
- `Day-6/`
- `Day-7/`

Each day can contain one or more independent projects with their own README, package files, and local setup instructions.

## Adding More Work

To add future work, create a new folder such as `Day-8/` and place one or more projects inside it. Dependency folders such as `node_modules/`, build outputs, logs, and local `.env` files are ignored from the root repository.
